Untuk memeriksa status firewall di Ubuntu dan melakukan pengaturan firewall, Anda dapat menggunakan program firewall standar yang disebut UFW (Uncomplicated Firewall). Berikut langkah-langkahnya:

### Cara Mengecek Status Firewall di Ubuntu (UFW):
1. Buka Terminal dengan menekan tombol `Ctrl + Alt + T` secara bersamaan.
2. Ketik perintah berikut untuk mengecek status firewall:
   ```
   sudo ufw status
   ```
   Jika UFW aktif, Anda akan melihat pesan yang menyatakan bahwa firewall sedang berjalan dan daftar aturan yang diterapkan.

### Cara Mengatur Firewall di Ubuntu (UFW):
1. **Menyalakan Firewall**:
   - Untuk menyalakan UFW, gunakan perintah:
     ```
     sudo ufw enable
     ```

2. **Menonaktifkan Firewall**:
   - Jika Anda ingin menonaktifkan UFW, gunakan perintah:
     ```
     sudo ufw disable
     ```

3. **Menambahkan Aturan (Rule)**:
   - Contoh: Untuk mengizinkan koneksi SSH (port 22), gunakan perintah:
     ```
     sudo ufw allow 22/tcp
     ```

4. **Menghapus Aturan (Rule)**:
   - Contoh: Jika Anda ingin menghapus aturan yang telah ditambahkan sebelumnya, gunakan perintah:
     ```
     sudo ufw delete allow 22/tcp
     ```

5. **Konfigurasi Lainnya**:
   - Anda juga dapat mengatur aturan firewall sesuai dengan kebutuhan spesifik Anda, seperti mengizinkan atau memblokir port tertentu.

Dengan menggunakan UFW, Anda dapat dengan mudah mengelola firewall di Ubuntu. Pastikan untuk memahami aturan yang Anda terapkan agar tidak mengganggu koneksi atau keamanan sistem Anda. Jika Anda memerlukan bantuan lebih lanjut atau memiliki pertanyaan lain, jangan ragu untuk bertanya!

Tentu! Berikut adalah panduan langkah demi langkah untuk menginstal OpenVPN, salah satu jenis VPN, di Ubuntu secara detail:

### Langkah 1: Buka Terminal
Buka terminal di Ubuntu. Anda dapat menemukan terminal dengan mencarinya di menu aplikasi atau dengan menekan tombol keyboard `Ctrl + Alt + T`.

### Langkah 2: Instal OpenVPN
Jalankan perintah berikut di terminal untuk menginstal OpenVPN:
```
sudo apt update
sudo apt install openvpn easy-rsa
```
Anda mungkin diminta untuk memasukkan kata sandi pengguna sudo.

### Langkah 3: Persiapkan Public Key Infrastructure (PKI)
Setelah instalasi selesai, Anda perlu mempersiapkan PKI untuk OpenVPN. Jalankan perintah berikut untuk memindahkan easy-rsa ke direktori yang sesuai:
```
make-cadir ~/openvpn-ca
cd ~/openvpn-ca
```

### Langkah 4: Konfigurasi Server OpenVPN
1. Buat direktori untuk menyimpan konfigurasi server:
```
./easyrsa init-pki
./easyrsa build-ca
./easyrsa gen-dh
./easyrsa gen-req server nopass
./easyrsa sign-req server server
```
2. Salin file konfigurasi server:
```
sudo cp ~/openvpn-ca/pki/dh.pem /etc/openvpn
sudo cp ~/openvpn-ca/pki/private/server.key /etc/openvpn
sudo cp ~/openvpn-ca/pki/issued/server.crt /etc/openvpn
sudo cp ~/openvpn-ca/pki/ca.crt /etc/openvpn
```
3. Buat file konfigurasi server:
```
sudo nano /etc/openvpn/server.conf
```
Tambahkan konfigurasi yang diperlukan dan simpan file.

### Langkah 5: Mulai Server OpenVPN
Jalankan perintah berikut untuk memulai server OpenVPN:
```
sudo systemctl start openvpn@server
sudo systemctl enable openvpn@server
```

### Langkah 6: Konfigurasi Klien
1. Buat sertifikat klien:
```
cd ~/openvpn-ca
./easyrsa gen-req client1
./easyrsa sign-req client client1
```
2. Salin sertifikat klien ke perangkat klien.

Dengan mengikuti langkah-langkah di atas, Anda dapat menginstal dan mengonfigurasi OpenVPN di Ubuntu. Pastikan untuk memperhatikan setiap langkah dengan cermat dan merujuk pada dokumentasi resmi OpenVPN atau tutorial terpercaya untuk bantuan lebih lanjut.

function simpanData(data) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // Cek jika header belum ada, tambahkan header
  if (sheet.getLastRow() == 0) {
    var headers = ["ID", "Hari", "Jam", "Kegiatan", "Keterangan", "Status"];
    sheet.appendRow(headers);
  }
  
  // Menentukan ID unik untuk setiap inputan
  var id = sheet.getLastRow();
  if (id === 1 && sheet.getRange('A2').getValue() === '') {
    id = 1; // jika baris kedua kosong, id tetap 1
  } else {
    id = id; // jika sudah ada data, id adalah jumlah baris terakhir
  }
  
  var rowData = [id, data.hari, data.jam, data.kegiatan, data.keterangan, data.status];
  
  // Menambahkan data ke baris berikutnya yang kosong
  sheet.appendRow(rowData);
}
